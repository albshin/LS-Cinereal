# LS-Cinereal

LS Cinereal was created in March 2013 and is a collaboration with DeviantArt user ymme1st and myself.

## Compatibility
To be used with a jailbroken iDevice with Winterboard installed from Cydia.

* iPhone 4
* iPhone 4S
* iPhone 5
* iPod Touch 4G
